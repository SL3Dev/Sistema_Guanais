Gestão de Dados e Backup

Importe dados de pacientes, atendimentos e financeiro através de arquivos Excel (.xlsx). O sistema processará automaticamente as relações entre os dados.

Importação e Exportação

Utilize os botões no cabeçalho para gerenciar backups dos seus dados:

  • Exportar: Gera um arquivo Excel com todos os dados do sistema.
  • Importar: Carrega dados de arquivos Excel (.xlsx, .xls, .csv).
API Backend

Este sistema utiliza uma API PHP para persistência de dados em banco MySQL. Os endpoints disponíveis são:

  • api/auth.php - Autenticação
  • api/pacientes.php - Gestão de pacientes
  • api/atendimentos.php - Gestão de atendimentos
  • api/financeiro.php - Gestão financeira
  • api/despesas.php - Gestão de despesas
